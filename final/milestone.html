The initial goal of our project was to render the volcano and also focus on the viscosity of the liquid; however, as we have progressed with our project, we realized that the focus should be primarily on the viscosity of the liquid.

The steps that we have accomplished in order to finish our project start with rendering the volcano, where we used project 4 as a baseline where the cloth mesh can be modified to simulate the volcano. By using the buildGrid method that was used for clothes in project 4, we were able to create a heightmap which would allow us to represent the height of the volcano. This is done through using a lighter color for the z-axis that is extracted from the png of the texture map.

The next step that we have made in this project was to render the lava, where the ultimate goal for the lava is to show it flowing down the volcano as if an eruption had just occurred. As of now, we have a volcano blob that is above our volcano using a lava texture; however, we will need to show this flowing down the volcano in our next steps.

Relative to our original progress plan, we believe that we are on the right track to finish our project and make the correct modifications, with a possibility to add on more features for our project if we have the time. As of now, we are also finalizing how we are going to use the Marching Cubes algorithm to help model a smooth mesh to help represent our viscous fluid. As the fluid moves down the volcano, we will need more and more particles to help show this, which can be done using the Marching Cubes algorithm while also maintaining the emission that we want to represent the essence of lava. We are planning to use the Marching Cubes algorithm specifically by computing a scalar value that is representative of the comprehensive lava that is around a specific position in the volcano.

In regards to our updated plan for the next two weeks, we will focus on finishing the lava flow and ensuring that we can maintain the emissive properties of the lava while also having it flow down the volcano. We will accompany this with testing so that we are able to ensure that our project is complete for the final presentation.
